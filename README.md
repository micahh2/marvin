Marvin
======

###1. Install Python 3.4 and VirtualEnv
>sudo apt-get install python3.4 virtualenv

###2. Setup up VENV
>virtualenv env -p /usr/bin/python3.4

###3. Activate VENV
>. env/bin/activate

###4. Install requirements
>pip install -r requirements3.txt

###5. Start Marvin
>python marvin.py

###6. Browse to: http://localhost:5000/

###7. Have fun.
